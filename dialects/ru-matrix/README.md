# COIL:ru-matrix — диалект «Красная таблетка»

> «Я не говорил, что будет легко. Я говорил, что будет правда.»

**Статус:** тематический community-диалект.

COIL — это код за поведением агентов. Матрица — код за реальностью людей. Метафора не натянута: она точна. Вы буквально пишете скрипт, который управляет тем, что происходит внутри симуляции.

Каждый оператор подобран так, чтобы его смысл в мире Матрицы отражал семантику COIL.

## Типы ПРОРОЧЕСТВА

| COIL | МАТРИЦА | Почему так |
|------|---------|------------|
| `ТЕКСТ` | `КОД` | Всё в Матрице — код. Текстовое значение = строка зелёного кода. |
| `ЧИСЛО` | `ЦИФРА` | Зелёные цифры, падающие по экрану. |
| `ФЛАГ` | `КРАСНАЯ/СИНЯЯ` | Красная = true (видишь правду). Синяя = false (остаёшься в неведении). Единственный булев тип, который не нужно объяснять. |
| `ВЫБОР(...)` | `ТАБЛЕТКА(...)` | «Ты выбираешь таблетку.» Enum = набор таблеток. «ТАБЛЕТКА(баг, фича, вопрос)» — выбери свою. |
| `СПИСОК` | `ПОТОК` | Зелёный поток символов. Массив = поток данных. |

## Сигнатуры ссылок

| Префикс | Значение | В мире Матрицы |
|----------|----------|---------------|
| `$` | Значение | «код» — данные внутри системы |
| `?` | Обещание | «пророчество» — предсказание Оракула, которое ещё не сбылось |
| `@` | Участник | «боец» — член экипажа |
| `!` | Инструмент | «оружие» — единица арсенала |
| `#` | Канал | «линия» — телефонная линия, мост между мирами |
| `~` | Поток | «зелёный дождь» — непрерывный поток данных |

## Пример: исследовательский агент

```coil
' ══════════════════════════════════════════════
' Экипаж на борту. Арсенал проверен.
' ══════════════════════════════════════════════
ЭКИПАЖ эксперт, автор
АРСЕНАЛ поиск

' ══════════════════════════════════════════════
' Загрузка навыков.
' «Я знаю кунг-фу.» — «Покажи.»
' ══════════════════════════════════════════════
ЗАГРУЗИ research_skill
<<
Ты — аналитик-исследователь.
Работаешь с неструктурированными запросами.
Всегда ищешь первоисточники и указываешь степень
уверенности в выводах. Избегаешь спекуляций.
Если данных недостаточно — говоришь об этом прямо.
>>
ПРОСНИСЬ

ЗАГРУЗИ domain_expertise
<<
Ты разбираешься в продуктовой аналитике.
Знаешь метрики retention, activation, churn.
Умеешь читать воронки и когортные отчёты.
При анализе всегда разделяешь корреляцию
и причинно-следственную связь.
>>
ПРОСНИСЬ

' ══════════════════════════════════════════════
' Входные данные.
' «Это твой последний шанс. Прими.»
' ══════════════════════════════════════════════
ПРИМИ запрос
<<
Запрос пользователя для исследования.
>>
ПРОСНИСЬ

ПРИМИ история_обращений
<<
История предыдущих обращений по этой теме.
>>
ПРОСНИСЬ

ПРИМИ кейс
<<
Идентификатор текущего кейса.
>>
ПРОСНИСЬ

' ══════════════════════════════════════════════
' 1. Прозрей.
'    «Не пытайся согнуть ложку.
'     Это невозможно. Вместо этого
'     попытайся понять правду:
'     ложки не существует.»
' ══════════════════════════════════════════════
ПРОЗРЕЙ анализ
  ЗНАЯ $research_skill, $domain_expertise
  БЕЛЫЙ КРОЛИК <<
  Подготовить обоснованный ответ с источниками.
  >>
  ЗАДАНИЕ <<
  Исследуй запрос пользователя. Учитывай все данные,
  которые поступят в процессе через поток.
  >>
  ДЕЖАВЮ <<
  Запрос: $запрос
  История обращений: $история_обращений
  >>
  ПРОРОЧЕСТВО
  * ответ: КОД - итоговый ответ
  * источники: ПОТОК - использованные материалы
    * ref: КОД - ссылка
    * why: КОД - почему релевантен
  * уверенность: ТАБЛЕТКА(высокая, средняя, низкая)
ПРОСНИСЬ

' ══════════════════════════════════════════════
' 2. Параллельно — взломать поиск
'    и передать эксперту.
' ══════════════════════════════════════════════
ВЗЛОМАЙ найденное
  ВООРУЖИВШИСЬ !поиск
  - query: $запрос
  - limit: 10
ПРОСНИСЬ

ПЕРЕДАЙ мнение
  НА ЛИНИЮ #консультации
  ВЫЗОВИ @эксперт
  ВЕРЬ ЛЮБОЕ ЗНАМЕНИЕ
  ИЛИ АГЕНТЫ 10м
  <<
  Нужна экспертиза по запросу: $запрос
  >>
ПРОСНИСЬ

' ══════════════════════════════════════════════
' 3. Глитчи в системе — данные поступают,
'    реальность обновляется.
' ══════════════════════════════════════════════
ЦИКЛ 2
  ЗАМРИ данные
    ПОКА НЕ СБУДЕТСЯ ?найденное, ?мнение
    ПУТЬ ЛЮБОЕ ЗНАМЕНИЕ
  ПРОСНИСЬ

  ГЛИТЧ ~анализ
    <<
    $данные
    >>
  ПРОСНИСЬ
ПРОСНИСЬ

' ══════════════════════════════════════════════
' 4. Замри. Пророчество сбывается.
' ══════════════════════════════════════════════
ЗАМРИ
  ПОКА НЕ СБУДЕТСЯ ?анализ
ПРОСНИСЬ

' ══════════════════════════════════════════════
' 5. Передай ответ автору.
'    «Я пришёл сюда, чтобы сказать,
'     чем всё это закончится.»
' ══════════════════════════════════════════════
ПЕРЕДАЙ
  НА ЛИНИЮ #результаты/$кейс
  ВЫЗОВИ @автор
  <<
  $анализ.ответ
  >>
ПРОСНИСЬ

ОТКЛЮЧИСЬ
```

## FAQ

**Q: Зачем?**
A: COIL позволяет писать скрипты на языке, на котором вы думаете. Некоторые люди думают на русском, некоторые — на корейском. А некоторые — цитатами из Матрицы. Этот диалект для них.

**Q: Почему именно Матрица?**
A: Потому что метафора точна, а не натянута. COIL — это код за поведением агентов. Матрица — код за поведением людей. Вы буквально пишете реальность. «ПРОЗРЕЙ анализ» значит то же, что Морфеус имел в виду: увидь правду за данными.

**Q: Это работает?**
A: Среда исполнения оперирует семантикой конструкций, не их написанием. Для рантайма `ПРОСНИСЬ` и `КОНЕЦ` — одно и то же. Для вас — нет.

**Q: КРАСНАЯ/СИНЯЯ — это правда булев тип?**
A: Красная таблетка = true = видишь правду. Синяя таблетка = false = остаёшься в неведении. Это единственный булев тип в истории программирования, который не нужно объяснять.

**Q: Другие тематические диалекты?**
A: Рассматриваются «Крёстный отец» (мафия — это язык оркестрации), «Кухня Рамзи» (FIRE! HEARD!) и другие. Также существуют [русский мат-диалект](../ru-mat/README.md) и [English profanity dialect](../en-profanity/README.md).
